How to use this app

What it is

This is a BBCode template generator for F95Zone-style release threads. You fill in structured fields for Game, Animation, or Comic / Stills, then produce the final BBCode from the output panel.

Tabs

Use the tabs at the top to switch between the three template types. Each tab has its own sections (metadata, links, downloads, tags, and so on).

Optional fields

Labeled chips above many fields turn optional blocks on or off. Only enabled fields are included when you generate BBCode.

Tags and languages

Add tags and languages from the searchable pickers. Click the × on a chip to remove it.

Generate and copy

  • Generate fills the right-hand output with BBCode from the current tab.
  • Copy copies that output to the clipboard (in the desktop app this uses the system clipboard).
  • Clear / Clear All reset output or the whole form, depending on the button.

Saving your work

The form auto-saves as you edit (tags, languages, mirrors, and other inputs are stored in localStorage under f95-bbcode-form-state-v2). Reloading the page restores the last state.

Settings

Settings stores your backend base URL (where the PHP helper is hosted) and your personal VNDB API token from vndb.org/u/tokens. The backend forwards lookups to VNDB using that token only.

VNDB does not accept your username and password on its JSON API. You sign in on the website once, create a token under My profile → Applications, and use that token like an API key.

Tools

  • Steam — Fetch & fill game tab: looks up a Steam App ID via the Store API, fills game fields, and caches the response on disk (desktop app).
  • Steam — Download screenshots: saves full-size store screenshots to your Downloads folder by default, or to a folder you pick. You can bundle them into a single ZIP.
  • VNDB lookup: calls your backend (POST /v1/vn/lookup) with an id or search string and applies the first result to the Game tab. Your VNDB token from Settings is sent with each request.

Steam and screenshot download run locally in the desktop app. VNDB lookup uses your saved backend URL and token.
